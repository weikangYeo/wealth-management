# Quick Note - Tailwind

- Flex: to make it responsible base on screen size
- p-<number>: padding https://tailwindcss.com/docs/padding
- px-<number>: inline padding (mean pad left & right)
- py-<number>: horizontal padding (pad up & down)
- overflow-<xxx>: what to do when an element is overflow (exceed the size)
- rounded-<size>: to control how round the element is, https://tailwindcss.com/docs/border-radius
- border border-<color>-<opacity>: border of element, and its color and opacity https://tailwindcss.com/docs/border-color
- w-<xxx>: width of element
